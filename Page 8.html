MY REFLECTION

In my video demonstration, I was able to show my understanding of several important CSS concepts such as float, overflow, display property, height and width, positions, and background images. While doing the activity, I learned how each property affects the layout and organization of elements on a webpage. I realized that float and display help control how items appear beside each other, while overflow manages content that goes beyond its container. Learning height and width taught me how to size elements properly, and exploring different position values helped me understand how to place objects exactly where I want them. Adding background images also helped me appreciate how design choices can improve the visual appeal of a webpage. Overall, this activity strengthened my skills in CSS and made me more confident in creating clean, well-structured, and visually appealing web designs.
